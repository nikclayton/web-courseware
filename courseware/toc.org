* Course structure

Students should work through this material in the following order

* Preparation weeks

This work can be done outside the classroom.

- *Week 1, getting familiar with tools* [[[file:../../slides/courseware/prep/week1/README.html][slides]]] [[[file:../../notes/courseware/prep/week1/README.html][notes]]]

  Setting up some initial tools and getting to know VSCode.

- *Week 2, initial HTML* [[[file:../../slides/courseware/prep/week2/README.html][slides]]] [[[file:../../notes/courseware/prep/week2/README.html][notes]]]

  Introduces HTML and researching material on the Mozilla Developer Network.

* The course

This work should be done in the classroom, and is intended to be done in the following order.

- *Introduction to the classroom* [[[file:../../slides/courseware/classroom/classroom-intro/README.html][slides]]] [[[file:../../notes/courseware/classroom/classroom-intro/README.html][notes]]]

  Initial material for day 1 in the classroom. You should update this to reference logistical information for your course.

- *Shell and Git* [[[file:../../slides/courseware/classroom/shell-and-git/README.html][slides]]] [[[file:../../notes/courseware/classroom/shell-and-git/README.html][notes]]]

  Recap the shell material from week 1. First steps with Git, including creating a GitHub account, configuring Git locally, and introductory Git workflow.

- *CSS* [[[file:../../slides/courseware/classroom/css/README.html][slides]]] [[[file:../../notes/courseware/classroom/css/README.html][notes]]]

  Initial CSS material.

- *Advanced CSS* [[[file:../../slides/courseware/classroom/advanced-css/README.html][slides]]] [[[file:../../notes/courseware/classroom/advanced-css/README.html][notes]]]

  Advanced CSS material

- *Introductory Javascript / DOM* [[[file:../../slides/courseware/classroom/javascript/introduction/README.html][slides]]] [[[file:../../notes/courseware/classroom/javascript/introduction/README.html][notes]]]

  Up to here is approximately 4 weeks worth of material depending on how quickly the students move through it.

- *Javascript Shopping List exercise* [[[file:../../slides/courseware/classroom/javascript/shopping/README.html][slides]]] [[[file:../../notes/courseware/classroom/javascript/shopping/README.html][notes]]]

  This is several weeks worth of project and exercises where we will
  build a frontend application from the ground up, applying the material
  that has been covered in the previous weeks.

- *Javascript Ecosystem* [[[file:../../slides/courseware/classroom/javascript/ecosystem/README.html][slides]]] [[[file:../../notes/courseware/classroom/javascript/ecosystem/README.html][notes]]]

  Introduction to tools used in the JS Ecosystem -- Node and NPM, JS and CSS minimisers, Grunt for task automation, watching files for changes, etc.

- *Javascript React* [[[file:../../slides/courseware/classroom/javascript/react/README.html][slides]]] [[[file:../../notes/courseware/classroom/javascript/react/README.html][notes]]]

  Introduction to React, building the shopping list using React. Also includes material on OO inheritance.

- *SQL* [[[file:../../slides/courseware/classroom/sql/README.html][slides]]] [[[file:../../notes/courseware/classroom/sql/README.html][notes]]]

  Introduction to SQL. This can be moved to other parts of the course or dropped, it's used if any of the internships require an intro to SQL.
