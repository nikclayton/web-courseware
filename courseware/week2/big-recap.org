* HTML

- HTML consists of nested *elements*
- Element: =<start_tag>content</end_tag>=
- Empty element: =<start_tag>=
- Elements can nest

#+BEGIN_EXAMPLE
<el-one>
  <el-two>Some content</el-two>
</el-one>
#+END_EXAMPLE
